## 📖 這是什麼？

**探索日記** 是一款專為旅遊愛好者設計的 **踩點記錄 App**。

你是否常在社群媒體上看到心動的餐廳或景點，卻在真正想去的時候找不到？  
探索日記讓你隨手收藏這些地方，造訪後留下記錄，讓每一段旅程都有跡可循。

🌐 **官方網站**：[https://xuanxuan-tech.github.io/Meow-Wanderer/](https://xuanxuan-tech.github.io/Meow-Wanderer/)
